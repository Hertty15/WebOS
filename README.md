# 🌟 My WebOS

A beautiful, functional web-based operating system built for the **NASA Stardance Challenge**!

![WebOS Screenshot](./screenshot.png)

## ✨ Features

- **🖥️ Draggable Windows** - Move windows around just like a real OS
- ** macOS-Style Dock** - Quick access to all your apps
- **🎨 Glassmorphism Design** - Modern, frosted-glass aesthetic
- **⏰ Live Clock** - Real-time clock in the top bar
- **📦 Multiple Apps:**
  - 🏠 Welcome (About Me)
  - 🧮 Calculator
  - 📝 Notes
  - 💻 Terminal
  - ⚙️ Settings
  - ℹ️ About

## 🚀 Try It Out

Visit the live demo: **[hertty15.github.io/WebOS](https://hertty15.github.io/WebOS/)**

## 🛠️ Built With

- HTML5
- CSS3 (with glassmorphism effects)
- JavaScript (for draggable windows and interactivity)

## 📖 How It Works

This project demonstrates:
- **DOM Manipulation** - Creating and managing window elements
- **Event Handling** - Mouse events for dragging and clicking
- **CSS Positioning** - Absolute positioning for windows and the dock
- **Responsive Design** - Works on different screen sizes

## 🎯 Project Goals

Built as part of the NASA Stardance Challenge to create a functional web operating system that showcases modern web development skills.
